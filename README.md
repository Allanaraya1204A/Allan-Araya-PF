# Personal Portfolio

This project is a personal portfolio built with Astro, CSS, and JavaScript. It showcases my work and projects, demonstrating my skills and abilities as a developer.

## Technologies Used
- **Astro:** A modern framework for building static websites.
- **CSS:** For styling and layout.
- **JavaScript:** For interactivity and dynamic content.

## Deployment
The portfolio is deployed on [Vercel](https://vercel.com), making it accessible online for potential employers and clients to view my work.

Feel free to explore the site and learn more about my projects!
